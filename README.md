#### Discord Bot Project Generator

### How to activate

run:
`npm i -g djs-bot-generation-package` THIS HAS TO BE GLOBAL

then:
`djs-bot-gen`

##### Description:

- The idea for this project is for people to quickly make a discord bot outline
